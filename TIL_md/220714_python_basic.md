- 데이터 저장

    2. 리스트 - 데이터를 여러개 저장하는 것(자바 배열)
    3. 딕셔너리 - 키, 맵 구조로 데이터를 저장(자바 해시맵)
- 조건문

    2. if / else
    3. if / elif / else
- 반복문

    2. for : 종료 조건이 필요없다.(반복 범위를 지정하므로)
